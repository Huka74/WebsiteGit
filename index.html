Brachistochrone
In this article, the famous Brachistochrone problem is solved using the Lagrangian formalism and the tautochrone property is derived.
An iteractive animation can be used to see the Brachistochrone curve for draggable starting and ending points. Three scenes can be toggled to compare the travel time of the Brachistochrone curve with custom spline curves, compare with a set of intiutive curves, and to see the tautochrone property in action.

Catenary
In this article, the shape of a hanging chain is derived using the Lagrangian formalism.
An interactive animation can be used to see how a chain reacts when moving its anchor points, and how it coincides with the theoretical shape derived.
